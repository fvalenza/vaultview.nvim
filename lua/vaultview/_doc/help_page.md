# Vaultview Help Page

## Keymaps

### Navigating the Vaultview
- `n` / `p`: Next / Previous board
- `1 -> 9`: Jump to board 1-9

- `<S-h>` / `<S-l>`: Previous / Next page

- `j` / `k`: Navigate down / up
- `h` / `l`: Navigate left / right
- `gg` / `G`: Go to top / bottom of the current list
- `<M-h>` / `<M-l>`: Go to the list most left/right of the page


### Actions

- `0`: Open help
- `q`: Close Vaultview
- `r`: Refresh current board  <!-- WIP -->

- `Enter`: Open selected item in neovim
- `o`: Open selected item in Obsidian

- `<C-f>`: picker for lists
- `f`: picker to entries
- `F`: picker for content into entries

- `x`: Toggle displaying of current entry
- `X`: Toggle displaying of current list
